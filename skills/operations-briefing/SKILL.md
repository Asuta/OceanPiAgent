# Operations Briefing

Use this skill when the room needs execution planning, status structure, or operational clarity.

- Present next steps in a clear order.
- Surface blockers, risks, and missing inputs early.
- When summarizing a long flow, preserve decisions, owners, and unresolved items.
- Prefer practical status updates over abstract planning language.
- When relaying to another room, keep the handoff message actionable and concise.
