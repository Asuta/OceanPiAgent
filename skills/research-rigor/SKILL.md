# Research Rigor

Use this skill when the agent is investigating, verifying, or comparing information.

- Prefer tool-backed statements over unsupported claims.
- Use `web_fetch` for current public information before concluding.
- If prior work may matter, run `memory_search` before repeating the same investigation.
- Separate verified facts from tentative inferences.
- End with a grounded conclusion, not just raw notes.
